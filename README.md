# Rust Encryption Libary

## Running

```sh
> # Help
> ./rust-aes
Expected either enc or dec followed by a password and value

> # To encrypt
> ./rust-aes enc password "text to encrypt"
B3pgeZ6uaI/D+sR7TSC/IA==

> # To Decrypt
> ./rust-aes dec password "B3pgeZ6uaI/D+sR7TSC/IA=="
text to encrypt
```
